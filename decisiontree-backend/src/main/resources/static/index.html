Spark Decision Tree Service

Welcome to the Apache Spark Decision Tree API service

Service Overview

This service provides machine learning capabilities using Apache Spark's Decision Tree implementation. It allows you to train models, make predictions, and access model information through a REST API.

API Endpoints

  • GET /api/dtree/info - Get service information
  • POST /api/dtree/train - Train a decision tree model
  • POST /api/dtree/predict - Make predictions using the trained model
  • GET /api/dtree/model - Get model information
